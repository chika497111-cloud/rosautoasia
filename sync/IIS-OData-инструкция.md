# Установка IIS + публикация OData для 1С:УНФ

## Часть 1. Установка IIS

### Шаг 1. Включить компоненты IIS

1. Нажми **Win + R**, введи `optionalfeatures` → **ОК**
2. Найди **"Службы IIS"** и раскрой дерево
3. Поставь галочки:

   **Компоненты разработки приложений:**
   - Расширения ISAPI
   - Фильтры ISAPI
   - ASP

   **Общие компоненты HTTP:**
   - Статическое содержимое
   - Документ по умолчанию
   - Ошибки HTTP

   **Средства управления:**
   - Консоль управления IIS

4. Нажми **ОК**, дождись установки (2-5 минут)
5. **Перезагрузи** компьютер
6. Проверка: `http://localhost` в браузере → страница IIS

### Шаг 2. Проверь модуль расширения 1С для веб-сервера

1. Запусти установщик платформы 1С → режим **"Изменить"**
2. Убедись что установлен **"Модуль расширения веб-сервера"**

---

## Часть 2. Права доступа

### Шаг 3. Дать права IIS_IUSRS

**На папку платформы:**
1. `C:\Program Files\1cv8\` или `C:\Program Files (x86)\1cv8\`
2. ПКМ → Свойства → Безопасность → Изменить → Добавить
3. Впиши `IIS_IUSRS` → Проверить имена → ОК
4. Полный доступ → Применить

**На папку базы данных:** то же самое

---

## Часть 3. Публикация

### Шаг 4. Конфигуратор ОТ ИМЕНИ АДМИНИСТРАТОРА

ПКМ на ярлык 1С → **Запуск от имени администратора** → Конфигуратор

### Шаг 5. Публикация

1. **Администрирование → Публикация на веб-сервере...**
2. Имя: `unf` (латинские буквы)
3. Веб-сервер: **Internet Information Services**
4. Поставь галочку **"Публиковать стандартный интерфейс OData"**
5. Нажми **Опубликовать**

### Шаг 6. Пул приложений (если 32-бит 1С на 64-бит Windows)

1. Win+R → `inetmgr` → Пулы приложений
2. Найди пул `unf` или `DefaultAppPool`
3. ПКМ → Дополнительные параметры
4. "Разрешить 32-разрядные приложения" → **True** (для 32-бит 1С)

---

## Часть 4. Настройка OData

### Шаг 7. Включить объекты

1. 1С:Предприятие → **Администрирование → Синхронизация данных → Настройки стандартного интерфейса OData**
2. "Загрузить метаданные" → поставь галочки на нужных объектах
3. Сохрани

---

## Часть 5. Проверка

Открой в браузере:
```
http://localhost/unf/odata/standard.odata
```

Логин/пароль — данные пользователя 1С.

Список товаров (JSON):
```
http://localhost/unf/odata/standard.odata/Catalog_Номенклатура?$format=json
```

---

## Типичные ошибки

| Ошибка | Решение |
|--------|---------|
| HTTP 500 | Права IIS_IUSRS, путь wsisapi.dll, 32/64-бит |
| HTTP 403 | Права пользователя 1С |
| Не открывается | Служба IIS не запущена (services.msc) |
| Публикация не выполнена | Конфигуратор не от администратора |
